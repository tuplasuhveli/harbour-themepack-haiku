# Haiku® theme pack

Haiku®-style theme pack (fonts + icons) for **[Muoto](https://openrepos.net/content/fravaccaro/muoto-ui-themer)** on Sailfish OS, shipped as a **companion app** (`harbour-themepack-haiku`).

Documentation: https://uithemer.github.io/harbour-muoto/docs/getstarted

## Build (Sailfish SDK)

```bash
sfdk config --push-target SailfishOS-<arch>
sfdk build
```

Install the RPM with Muoto 3.x, then apply the **Haiku®** theme from Muoto settings.

## Builds

RPM builds run on GitHub Actions (`.github/workflows/main.yml`) for Sailfish OS 5.0 (`aarch64`, `armv7hl`, `i486`).

- **OpenRepos:** https://openrepos.net/content/fravaccaro/harbour-themepack-haiku
- **GitHub Releases:** https://github.com/uithemer/harbour-themepack-haiku/releases

Publishing a GitHub Release attaches RPMs from CI; upload the same RPMs to OpenRepos (see `fonts/assets/openrepos-blueprint-haiku.md` in the uithemer fonts workspace).

## Icons

SVG sources live under `theme/*/scalable/`. Convert HVIF with [hvif-tools](https://github.com/threedeyes/hvif-tools) `icon2icon`:

```bash
# one-time: clone & build (or use .cache/hvif-tools)
git clone --recursive https://github.com/threedeyes/hvif-tools.git .cache/hvif-tools
cmake -S .cache/hvif-tools -B .cache/hvif-tools/build -DCMAKE_BUILD_TYPE=Release
cmake --build .cache/hvif-tools/build

./scripts/hvif_to_svg.sh   # *.hvif → *.svg in theme/*/scalable/
```

Download pinned icons from [hvif-store.art](https://hvif-store.art) (catalog API: `api.php`):

```bash
curl -sL 'https://hvif-store.art/api.php?search=FAQ'   # tutorial launcher
curl -sL -o .cache/hvif-store/faq.hvif \
  'https://hvif-store.art/uploads/icon_6942414d12969.hvif'
```

Export PNGs for Muoto:

```bash
cd theme
./themepack-helper.sh   # requires Inkscape 1.x
```

### Overlays (unthemed app backgrounds)

Muoto composites `theme/overlay/*.png` behind stock icons that your pack does not replace. Six circular gradient plates (`overlay01`–`overlay06`) use Haiku® desktop colors (blue, beige, selection blue, orange, teal, grey). They are **generic backgrounds only** — no HAIKU® logo, leaf mascot, or other trademark artwork.

Sources: `theme/overlay/overlayNN.svg` (generated by [`scripts/generate_overlays.py`](scripts/generate_overlays.py), based on the Numix Circle plate structure). Regenerate SVGs and PNGs:

```bash
python3 scripts/generate_overlays.py
cd theme && ./themepack-helper.sh
```
